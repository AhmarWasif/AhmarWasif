# Hi, I'm Ahmar 👋

I find the repetitive, judgment-heavy work operators actually do — research, analysis, copywriting — and build the system that does it. Same instinct I bring at Deloitte, shipped solo.

### Selected builds

**[Company Brief](https://github.com/AhmarWasif/company-brief-extension)** — Account and company research is the repetitive prep work that quietly eats an operator's week. This kills it: point it at any company's website and it reads the pages, runs live searches, and returns a structured brief in under a minute — pre-meeting prep, account research, or fast diligence, done. Under the hood it's a multi-step agent with fallback handling so it doesn't break on messy real-world sites. 

**[Venture Funnel](https://github.com/AhmarWasif/vc-funding-dashboard)** — Operator analytics, end to end: take 1,200 AI/SaaS funding rounds and build the layer that lets anyone actually interrogate them — live filtering plus a plain-English "ask the data" interface. The part I'm proudest of is what it won't do: it refuses questions the underlying data can't honestly answer, because an analysis leadership can't trust is worse than no analysis. Messy data in, defensible decisions out. 

**[Cover Letter Generator](https://github.com/AhmarWasif/cover-letter-generator)** — An AI writing tool I built to get the production layer right: secure API-key handling, hard cost controls, and usage governance enforced at the code level — not bolted on after. The unglamorous infrastructure that decides whether an AI feature ships safely and on budget, and the same spend-and-governance discipline I bring to enterprise AI.
